## Problem
Write a Dart function that takes a **list of integers** and returns a **new list with all duplicate elements removed**, while **keeping the original order** of elements.

### Rules
- Do **not** use built-in methods like `.toSet()`
- The original list should **not be modified**
- The list will **not be empty**

## Example

Input: [1, 2, 2, 3, 4, 3, 5]  
Output: [1, 2, 3, 4, 5]

Input: [5, 5, 5, 1, 2]  
Output: [5, 1, 2]
