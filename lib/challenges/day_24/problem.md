## Problem
Write a Dart function that takes a **list of integers** and returns a **new list sorted in ascending order**.

### Rules
- Do **not** use built-in sorting methods like `.sort()`
- The original list should **not be modified**
- The list will **not be empty**

## Example

Input: [4, 2, 7, 1, 3]  
Output: [1, 2, 3, 4, 7]

Input: [5, 1, 5, 3]  
Output: [1, 3, 5, 5]
