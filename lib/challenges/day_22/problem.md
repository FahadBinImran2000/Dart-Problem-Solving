## Problem
Write a Dart function that takes a **string** as input and returns a **map** showing how many times each character appears in the string.

- Consider **all characters** (letters, spaces, symbols).
- The string will **not be empty**.

## Example

Input: "hello"

Output:
{
h: 1,
e: 1,
l: 2,
o: 1
}

Input: "aab c"

Output:
{
a: 2,
b: 1,
" ": 1,
c: 1
}
