## Problem
Write a Dart function that takes a **sentence (string)** as input and returns the **longest word** in the sentence.

- If there are multiple words with the same maximum length, return **any one of them**.
- Words are separated by **spaces**.
- The sentence will **not be empty**.

## Example

Input: "I am learning flutter development"  
Output: "development"

Input: "dart is fun"  
Output: "dart"

Input: "code well"  
Output: "code" or "well"
