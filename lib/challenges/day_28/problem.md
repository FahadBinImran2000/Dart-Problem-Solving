## Problem
Write a Dart function that takes **two lists of integers** and returns a **new list containing only the elements that appear in both lists**.

### Rules
- The order of elements should follow the order of the **first list**
- Each element should appear **only once** in the result
- Do **not** modify the original lists
- Do **not** use built-in methods like `.toSet()`

## Example

Input:  
List1 = [1, 2, 3, 4, 5]  
List2 = [3, 4, 4, 5, 6]

Output:  
[3, 4, 5]

Input:  
List1 = [7, 8, 9]  
List2 = [1, 2, 3]

Output:  
[]
