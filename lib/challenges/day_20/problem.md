## Problem
Write a Dart function that takes a list of integers and returns the **number that appears least frequently** in the list.

If multiple numbers have the same lowest frequency, return **any one of them**.

The list will **not be empty**.

## Example
Input: [1, 3, 2, 1, 4, 1, 3]  
Output: 2 or 4

Input: [5, 5, 6, 6, 7]  
Output: 7
